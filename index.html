Recent Projects

Gas Mileage Calculator

My second project at General Assembly a Gas Mileage Calculator, this project features a client built in JavaScript, HTML and CSS/Sass, with additional Bootstrap components and using Handlebars for detail entries. The API is a Ruby on Rails implementation on top of a PostgreSQL relational database and deployed on Heroku.

Cagusabi Image Uploader

This is my fork of our group project at General Assembly, an image uploader. With a client built in JavaScript, HTML and CSS/Sass, this app makes extensive use of Bootstrap modals. The API was built in Express.js, with Mongoose.js on top of MongoDB. The API is deployed on Heroku, utilizing buckets on Amazon Web Services for storing images.

Tic Tac Toe

My first project at General Assembly. The client is built in JavaScript, HTML, CSS/Sass and Bootstrap. The API used was supplied by General Assembly. In the near future, the GA-provided API will be supplanted with a bespoke API using Express.js, Mongoose.js and MongoDB.
